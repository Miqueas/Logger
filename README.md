# Logit

Dependency-free, cross-platform and small logging library for Lua and Nim, with a simple and comfortable API.

![logit capture](capture.png)

## Installation

For Nim, just:

```
nimble install logit
```

## Documentation

For Nim, check out [this](NimDocs.md).
